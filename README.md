DESCRIÇÃO DO PROJETO 
Locadora de filmes – Desktop System  
O sistema é uma plataforma de Locadora de Filmes desenvolvida para gerenciar o aluguel de 
t
ítulos e o controle de usuários. O projeto foca em oferecer uma interface intuitiva tanto para o 
cliente final quanto para o gestor da locadora. 
Níveis de Acesso 
O sistema possui dois perfis distintos de utilização: 
• Perfil Usuário (Cliente): 
• Visualização do catálogo completo de filmes. 
• Acesso a valores de aluguel e categorias. 
• Realização de locação com datas de devolução. 
• Perfil Administrador (ADM): 
• CRUD de Filmes: Adicionar, editar e excluir títulos do catálogo. 
• Gestão de Usuários: Criar, editar e remover contas de acesso. 
• Manutenção: Controle total das tabelas do sistema. 
TECNOLOGIAS E FERRAMENTAS 
Desenvolvimento e Frameworks 
• C#: Linguagem principal para a lógica do sistema. 
• .NET: Plataforma de desenvolvimento utilizada. 
• Windows Forms (WinForms): Framework para criação da interface gráfica desktop. 
• Visual Studio Community 2022: IDE completa utilizada para a codificação e depuração 
do projeto. 
Banco de Dados 
• MySQL Server: Sistema de gerenciamento de banco de dados relacional para 
persistência das informações. 
Design e Comunicação Visual 
• Figma: Utilizado para a prototipagem de alta fidelidade das telas (UI/UX). 
• Canva: Ferramenta auxiliar para a criação de elementos gráficos e identidade visual do 
projeto. 
Versionamento e Terminal 
• GitHub: Plataforma de hospedagem e colaboração do código-fonte. 
• GitBash: Emulador de terminal utilizado para execução dos comandos Git de forma ágil.
